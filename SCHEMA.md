## Collection types

Note: 
Months are represented with the first day of the month (i.e. 01-01-2022). 
Do not add multiple rows for the same month on these tables.

### BalanceHistory
- **Month**: Date (date)
- **Total**: Number (Decimal)

### ForexBalanceHistory
- **Month**: Date (date)
- **Total**: Number (Decimal)

### ProfitLoss
- **Month**: Date (date)
- **Profit**: Number (Decimal)
- **Loss**: Number (Decimal)

### Transaction
- **Date**: Date (date)
- **Name**: Text (short)
- **Amount**: Number (Decimal)

### Wire
- **Name**: Text (short)
- **Amount**: Number (Decimal)
